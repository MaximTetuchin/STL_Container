Detailed Description

template<typename T, typename Allocator = std::allocator<T>>
class stl_containers::CircularList< T, Allocator >

A circular doubly-linked list container implementation in STL style.

Template Parameters
TType of elements stored in the list
AllocatorAllocator type for memory management (default: std::allocator<T>)

This class provides a circular doubly-linked list implementation with full STL compatibility. It supports bidirectional iterators, exception safety, and follows RAII principles.
